I'm excited to share that I've completely overhauled the software and
documentation for our LEGO Brick Sorting Machine! After many hours of tinkering
and testing, I'm confident these updates will make your sorting experience even
better.

**What's New**:

* **Updated Software**: The new version is now completely
  up-to-date and using the latest libraries. The previous version did not run
  on many modern Raspberry Pi's.
* **Revised Documentation**: The updated documentation now
  provides clear, step-by-step instructions for installation and setup, and
  troubleshooting. All combined, I think there is about 10 times more
  documentation now.

**﻿﻿How to Update**:

Everything is available on Github: <https://github.com/pbackx/lego-sorter-pi>

replying to this mail.
